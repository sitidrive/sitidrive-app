# sitidrive-app 🚗
sitidrive — a zone-based shared mobility platform for Abuja/City commuters. Helping people move smarter with reliable, affordable shared rides.


## 🚀 What It Does

- Connects riders and drivers along fixed city routes
- Uses familiar pickup zones (e.g. Shoprite Lugbe, Berger)
- Enables affordable, shared ride experince with predictable pricing

## 🎯 Problem

Daily commuting in Abuja is:
- Unpredictable
- Stressful
- Inefficient

SititDrive organizes existing transport behavior into a reliable system.

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- Firebase (Auth + Firestore)

## ⚡ Current Status

MVP in development and real-world testing phase.

- Lugbe → Berger
- Lugbe Shoprite Bridge → Central Area
